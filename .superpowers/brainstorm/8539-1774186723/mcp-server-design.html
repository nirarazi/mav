MCP Server Design: Tool Surface Area

Since agent-protocol-native is Maverick's core moat, the MCP server is the most critical component. What's the right tool surface?

Proposed MCP Tool Categories

Click each category to select which ones belong in MVP vs. later phases.

1

Content Management (MVP)

create_post, list_posts, update_post, delete_post, upload_media

Core CRUD for posts across platforms. An agent can draft, schedule, edit, and delete social media content. This alone makes Maverick useful.

2

Account & Platform (MVP)

list_integrations, get_platform_rules, get_account_status

Discovery tools — agent learns what platforms are connected, what constraints each has (char limits, media types, rate limits), and account health.

3

Approval Workflow (MVP)

get_pending_approvals, approve_item, reject_item, add_feedback

Enables an orchestrator agent (Paperclip CEO) to review and approve content from the social media agent. Critical for Guardian mode.

4

Persona (MVP)

list_personas, get_persona, set_active_persona, create_persona

Agent can discover available brand voices, switch between them, and create new ones. Essential for multi-brand zero-human companies.

5

Autonomous Brain (Phase 2)

run_content_cycle, set_goal, get_goal_progress, generate_weekly_plan

Trigger the autonomous content generation cycle. Set strategic goals. Let a Paperclip CEO agent say "post about our new product launch" and the brain handles everything.

6

Analytics & Insights (Phase 2)

get_analytics, get_engagement_summary, get_brand_sentiment, get_best_times

Agent can query engagement metrics, understand what's working, and make data-driven content decisions. Feeds back into the strategy layer.

7

Engagement & Replies (Phase 2)

get_mentions, reply_to_mention, get_dms, triage_dm, get_comments

Two-way social media interaction. Agent monitors mentions, generates contextual replies, and triages DMs. Turns Maverick from a poster into a communicator.

8

Compliance & Audit (Phase 2-3)

get_compliance_report, get_audit_log, check_content_compliance, get_cost_report

Transparency tools. Agent can verify compliance status, pull audit trails, and check cost budgets. Enterprise moat builder.

MCP Resources (Read-only context for agents)

persona://active Current persona config (voice, rules, constraints) MVP
platform://rules/{id} Platform-specific constraints and ToS rules MVP
queue://approvals Pending approval items with content preview MVP
analytics://summary Recent engagement metrics dashboard Phase 2
goals://active Current strategic goals and progress Phase 2

MVP Tool Count: 17 tools + 3 resources

Categories 1-4 give agents enough capability to manage a full social media presence through Maverick from day one: discover accounts, create content with the right persona, schedule posts, and handle the approval workflow.
