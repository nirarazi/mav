Agent Brain Architecture: Which Approach?

The agent brain is what makes Maverick autonomous. How should it be built?

A

LangGraph State Machine

Graph-based workflow where each node is a step (ideate, draft, comply, schedule). Edges are conditional transitions. Built-in interrupt() for human-in-the-loop pauses.

Pros

  • Production-proven (LinkedIn, Uber)
  • Native HITL with checkpointing
  • Visual debugging of execution flow
  • Handles complex branching (retry, escalate, skip)

Cons

  • Heavy dependency (LangChain ecosystem)
  • Learning curve for contributors
  • Overkill if workflows stay simple
  • Python-first ecosystem (we're TypeScript)
B

Simple Pipeline + Temporal

Linear pipeline of TypeScript functions orchestrated by Temporal (which Postiz already uses). Each step is a Temporal activity. HITL = Temporal signal/query. No AI framework dependency.

Pros

  • Zero new dependencies (Temporal already in Postiz)
  • Pure TypeScript, no framework lock-in
  • Battle-tested durability and retries
  • Easy for any TS dev to understand

Cons

  • No built-in LLM orchestration patterns
  • Manual implementation of branching logic
  • Less visibility into AI reasoning
  • HITL requires custom signal handling
C

Hybrid: Vercel AI SDK + Temporal

Use Vercel AI SDK (TypeScript-native, LLM-agnostic) for the AI calls and tool use. Temporal for workflow durability and scheduling. Combine both strengths without heavy framework.

Pros

  • TypeScript-native AI SDK with streaming
  • Built-in provider abstraction (any LLM)
  • Temporal for durability + scheduling
  • Aligns perfectly with BYOK requirement

Cons

  • Two systems to understand
  • No built-in graph visualization
  • HITL still needs custom implementation
  • Vercel AI SDK less battle-tested than LangGraph

Recommendation: C (Hybrid)

Given that Maverick is TypeScript-first (NestJS/Next.js), BYOK is a requirement, and Temporal is already in the stack, the hybrid approach gives us the best of both worlds without pulling in a heavy Python-oriented framework. The Vercel AI SDK handles LLM abstraction elegantly, and Temporal gives us the durability we need for scheduled autonomous cycles.
